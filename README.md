# 🎬 Movie Recommendation System

A content-based movie recommendation system that suggests similar movies based on genres using Machine Learning techniques.

## 📌 Features

- Content-based filtering
- Genre similarity matching
- Personalized recommendations
- Fast recommendation engine

## 🛠 Tech Stack

- Python
- Pandas
- Scikit-Learn
- Machine Learning
- Cosine Similarity

## 📂 Project Structure

movie-recommendation-system/

├── movie_recommendation.py

├── requirements.txt

├── README.md

└── screenshots/

    └── output.png

## 🚀 Installation

```bash
pip install -r requirements.txt
```

```bash
python movie_recommendation.py
```
